# Sistema de Avisos - Etec São Sebastião

Este projeto é uma plataforma digital desenvolvida para centralizar e exibir comunicados, eventos e notícias importantes diretamente na entrada da Etec de São Sebastião. O objetivo é garantir que alunos, professores e visitantes tenham acesso rápido às informações essenciais assim que adentrarem a instituição.

## Sobre o Projeto

O sistema foi concebido para substituir murais de papel por uma solução digital dinâmica e sustentável. Ele é dividido em duas frentes principais:

### Painel Administrativo:
Uma interface restrita para a criação, edição e exclusão de avisos.

### Carrossel de Exibição:
Uma interface visual otimizada para ser exibida em monitores/TVs na entrada da escola, alternando os avisos automaticamente.


## Funcionalidades

Gestão de Avisos: Criação de recados com título, conteúdo e (opcionalmente) imagens.

Exibição em Tempo Real: Carrossel responsivo que atualiza as notícias conforme são publicadas.

Filtro de Relevância: Possibilidade de definir por quanto tempo um aviso deve ficar no ar.

Design Intuitivo: Interface limpa seguindo a identidade visual da instituição.
## Tecnologias Utilizadas

Frontend: HTML5, CSS3, JavaScript.

Backend: PHP.

Banco de Dados: MySQL

Estilização: Bootstrap 5.
